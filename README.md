Astro Sbmn7 is a **Python-based Vedic astrology application** that calculates charts, Dashas, Panchang, and more.  
It is designed for users who want to test and explore astrology using Python without compilation.

---

## Features

- D1 to D150 divisional charts
- Vimshottari and other Dasha calculations
- Panchang: Tithi, Nakshatra, Yoga, Karana
- Gochar (transits) analysis
- Shadbala and planetary strength calculations
- Full Swiss Ephemeris support (`ephe` folder included)
- Moon phases and fonts included
